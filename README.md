# Шаблон LaTeX для кафедры ИТАС

_(no English readme, sorry)_

Подходит под требования СТП 2024 кафедры Информационных автоматизированных систем обработки информации.

Для сборки используется Docker и Makefile. В README предполагается, что окружение - Linux\MacOS с нужными правами для Docker + доступен `make`.

## Использование

1. Клонируйте репозиторий

```bash
git clone https://github.com/kawunus/bsuir-itas-stp-latex 
```

2. Соберите контейнер

```bash
docker buildx build -t kawunus:latex .  
```

3. Соберите документ

```bash
make pvc
```

Эта команда собирает новый артефакт работы и помещает его рядом с Makefile'ом (естественно, запускать нужно находясь в одной папке с Makefile). Автоматически подтягивает сохраненные изменения

Или для единоразовой сборки документа:

```bash
make note.pdf
```

## Примеры, кастомные функции и дополнительная информация

### Структура проекта

В файле `title.tex` содержится титульный лист для работы.

Файл `table_of_contents.tex` автоматически генерирует содержание.

В файле `note.tex` описывается струтура проекта.

Для курсовых так же предусмотрен файл `task.tex`, в котором содержится задание к курсовой работы.

### Категории

```tex
\crsection{Название категории} % Секция для курсовых работ. Начинается с новой страницы
```

```tex
\lbsection{Название категории} % Секция для лабораторных работ. Продолжает с той же страницы.
```

```tex
\anonsection{Название категории} % Анонимная секция для курсовых работ. Продолжает с той же страницы.
```
